Politique de confidentialité

Dernière mise à jour : 13 mars 2026

1. Responsable du traitement

Le responsable du traitement des données est Corentin, éditeur de Neoskool, domicilié dans le Canton de Vaud, Suisse.

Contact : contact@neoskool.com

2. Cadre juridique

Le traitement de vos données est soumis à :

  • La Loi fédérale sur la protection des données (LPD) — droit suisse
  • Le Règlement général sur la protection des données (RGPD) — pour les utilisateurs de l'UE/EEE

4. Données NON collectées

Neoskool ne collecte pas :

  • De données sur les élèves (noms, notes, résultats)
  • De cookies de tracking ou publicitaires
  • De données de géolocalisation
  • De données biométriques

6. Durée de conservation

  • Compte actif : données conservées tant que le compte existe
  • Suppression de compte : toutes les données sont supprimées sous 30 jours
  • Données de paiement : conservées par Stripe selon leurs obligations légales (10 ans pour la comptabilité)

7. Vos droits

Conformément à la LPD et au RGPD, vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données personnelles
  • Droit de rectification : corriger des données inexactes
  • Droit à l'effacement : supprimer votre compte et toutes vos données
  • Droit à la portabilité : recevoir vos données dans un format structuré
  • Droit d'opposition : vous opposer au traitement de vos données
  • Droit de retrait du consentement : retirer votre consentement à tout moment

Pour exercer ces droits, contactez-nous à contact@neoskool.com. Nous répondons sous 30 jours.

8. Suppression de compte

Vous pouvez supprimer votre compte à tout moment depuis votre espace personnel (Mon compte → Supprimer mon compte). La suppression entraîne :

  • L'effacement de votre profil et identifiants
  • L'effacement de toutes vos fiches sauvegardées
  • L'effacement de vos tags, notes et préférences
  • L'annulation de tout abonnement actif

Cette action est irréversible.

9. Sécurité

Nous mettons en place les mesures suivantes pour protéger vos données :

  • Chiffrement HTTPS (TLS 1.3) pour toutes les communications
  • Mots de passe hashés (bcrypt) — jamais stockés en clair
  • Row Level Security (RLS) sur la base de données — chaque utilisateur n'accède qu'à ses propres données
  • Authentification sécurisée via Supabase Auth
  • Aucun accès tiers aux données utilisateur

10. Cookies

Neoskool utilise uniquement des cookies techniques essentiels :

  • Cookie de session Supabase (authentification)
  • LocalStorage pour les préférences d'interface

Aucun cookie publicitaire, analytique ou de tracking n'est utilisé.

11. Intelligence artificielle

Les fiches sont générées par l'API Claude (Anthropic). Les données envoyées à l'API pour la génération sont :

  • La matière, le degré, la voie et le sujet choisis
  • Les adaptations BEP sélectionnées
  • Le message personnalisé de l'utilisateur

Aucune donnée personnelle (email, nom, identifiants) n'est transmise à l'API de génération. Anthropic ne conserve pas les données des appels API au-delà de 30 jours.

12. Mineurs

Neoskool est destiné aux enseignants et parents. Le service ne collecte pas directement de données auprès de mineurs. Les fiches générées sont destinées à être utilisées dans un cadre pédagogique supervisé.

13. Modification de cette politique

Nous nous réservons le droit de modifier cette politique. Toute modification sera publiée sur cette page avec une date de mise à jour. Les changements significatifs seront notifiés par email.

14. Contact et réclamation

Pour toute question ou réclamation relative à vos données :

  • Email : contact@neoskool.com

En cas de litige non résolu, vous pouvez déposer une plainte auprès du Préposé fédéral à la protection des données et à la transparence (PFPDT) en Suisse, ou de l'autorité de protection des données de votre pays de résidence.
